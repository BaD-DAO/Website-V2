# BuildaDAO Website & Sub-Websites

This repository contains the source code for the main BuildaDAO website and its related sub-websites, built with Next.js.

## Project Overview

This project is structured to contain several key web properties related to the BuildaDAO ecosystem. The goal is to provide a central hub for all information, services, and community activities.

## Website Structure

The website is divided into the following main sections:

### 1. Main BuildaDAO Website

*   **Purpose**: To provide comprehensive information about BuildaDAO, its mission, vision, and how to get involved.
*   **Content**:
    *   About BuildaDAO
    *   Core Team
    *   Whitepaper
    *   Roadmap
    *   Contact Information

### 2. The First 500

*   **Purpose**: A dedicated section to showcase and track the progress of the first 500 startups incubated and accelerated through BuildaDAO.
*   **Content**:
    *   List of startups.
    *   Individual startup profiles with detailed progress, milestones, and funding status.
    *   Success stories and case studies.

### 3. BaD Academy

*   **Purpose**: The educational arm of BuildaDAO, offering courses in cutting-edge technologies.
*   **Content**:
    *   **AI Courses**:
        *   Junior Level: Introduction to AI, Machine Learning basics.
        *   Enterprise Level: Advanced AI for business, custom AI solutions.
    *   **Blockchain Courses**:
        *   Junior Level: Blockchain fundamentals, smart contract development for beginners.
        *   Enterprise Level: Advanced DeFi, enterprise blockchain implementation, security audits.
    *   Course catalog, instructor profiles, and enrollment information.

### 4. The BaD Countries

*   **Purpose**: To highlight and manage the activities of BuildaDAO's presence in various countries.
*   **Content**:
    *   Dedicated pages for each country chapter:
        *   BaD Ethiopia
        *   BaD Ghana
        *   BaD Cameroon
        *   BaD Uganda
        *   BaD Botswana
    *   Local initiatives, news, and contact information for each chapter.

### 5. Events

*   **Purpose**: To manage and promote all events related to BuildaDAO.
*   **Content**:
    *   **Upcoming Events**: A calendar of future workshops, webinars, and conferences.
    *   **Event Organizing Arm**: Information about our event management services for other organizations.
    *   **Event Ambassadors**: Program details for community members who want to represent BuildaDAO at events.

### 6. The DAO Talent Pool

*   **Purpose**: A platform to connect skilled professionals from the DAO with projects and opportunities.
*   **Content**:
    *   Profiles of experts in:
        *   Cybersecurity
        *   Design (UI/UX)
        *   Development (Frontend, Backend, Smart Contracts)
        *   Marketing & Growth Hacking
    *   How to join the talent pool.
    *   How to hire talent from the pool.

## Getting Started with Next.js

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
