# Design Reference from index (1).html

## Key Design Elements Observed:

### Color Scheme:
- **Primary**: Blue (#2562FE)
- **Background**: Dark/Black
- **Text**: White primary, gray secondary
- **Accents**: Blue gradients

### Typography:
- Large hero titles
- Clean sans-serif font
- Text hierarchy with different sizes (text-sm, text-xl, text-2xl, h1, h2)

### Layout Sections:
1. **Hero Section** - Large title "Tech Nexus Summit 2025" with location and date
2. **Meet Our Speakers** - Swiper carousel with speaker cards
3. **Impact/Stats Section** - Number counters with achievements
4. **Timeline/Itinerary** - Vertical timeline with event schedule
5. **Event at a Glance** - Grid of features with icons
6. **Pricing** - Two-tier pricing cards (General $49, VIP $99)
7. **Testimonials** - Swiper carousel with customer reviews
8. **CTA Section** - Call to action
9. **Contact Form** - Contact information and form
10. **Footer** - Links and newsletter signup

### Key Features:
- **Swiper.js** for carousels
- **GSAP animations** for scroll effects
- **Number counters** for statistics
- **Gradient overlays** on images
- **Hover effects** on links and buttons
- **Responsive design** with breakpoints

### Button Styles:
- Primary buttons with clip animations
- Secondary outlined buttons
- Hover state transitions

### Navigation:
- Sticky navbar
- Mobile hamburger menu
- Smooth scroll to sections
